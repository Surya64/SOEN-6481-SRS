This Contains the Implementation of below User Story.

User Story :
"As a user, I want to be able to see a summary of my order before finalizing my purchase, so that I can confirm correctness"

Folder Contains:
- Source Code
- Implementation Doc (TVM_TC_003)
- Executable JAR file
- Acceptance Test Results


How to run the Program:
- Double Click on the Executable JAR file(D2.jar)

or

- Open CMD
- Go to your ".jar" file directory
- Command : java -jar jarfilename.jar  (java -jar D2.jar)
